# Iris Flower Classification

## Project Description
This project uses Machine Learning to classify iris flowers into:
- Setosa
- Versicolor
- Virginica

based on flower measurements.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Features
- Data Loading
- Model Training
- Prediction
- Accuracy Calculation

## Output
The model predicts flower species with high accuracy.
